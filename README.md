## 📝 Sobre o Projeto

O **V-ERP** é uma plataforma de planejamento de recursos empresariais focada na centralização de processos e eficiência operacional. O sistema oferece autenticação segura, persistência de dados em banco de dados relacional e uma interface moderna para o usuário final.


---

## ✨ Funcionalidades

- **🔐 Autenticação:** Login seguro validado via API interna em PHP.
- **📊 Gestão de Dados:** Interface preparada para controle total de informações empresariais.
- **📱 Interface Fluida:** Uso de CSS e JS para garantir uma navegação rápida e responsiva.
- **📂 Arquitetura Limpa:** Separação lógica entre Front-end e Back-end.
